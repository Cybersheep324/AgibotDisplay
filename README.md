# AgibotDisplay

AgibotDisplay 是一个面向公开使用场景发布的字体项目。

**你可以免费使用、商用、嵌入、分发、修改和再发布本字体。**  
**你也可以在修改后继续使用原名称 “AgibotDisplay”。**

但请注意：

**本项目开放的是字体文件本身的使用权，不代表授予任何与“智元创新”相关的品牌、商标、Logo 或官方身份权利。**

---

## 项目特点

- **免费使用**
- **允许商业使用**
- **允许 WebFont / App / 软件嵌入**
- **允许上传字体站二次分发**
- **允许单独传播字体文件**
- **允许修改和再发布**

---

## 你可以做什么

在遵守 `LICENSE` 的前提下，你可以：

- 用于个人项目
- 用于商业项目
- 用于海报、包装、广告、网页、App、视频、演示等场景
- 嵌入网站、App、软件、电子文档或终端设备
- 上传到字体站、素材站、代码仓库、镜像站等平台免费传播
- 修改、补字、做衍生版本
- 再发布原版或修改版本
- 修改后继续使用名称 **AgibotDisplay**

---

## 你不能做什么

你**不能**：

- 将字体文件本身作为收费商品出售或转售
- 以本字体的发布或使用为由，暗示自己与 **智元创新** 存在官方合作、授权、认证、背书、赞助或其他关联关系
- 将本项目表述为 **智元创新官方品牌资产全面开放授权**

---

## 关于名称与修改版本

本项目**允许修改后继续使用原名**。  
这意味着社区版本、扩展版本、补字版本或其他修改版本，可以继续使用 **AgibotDisplay** 这一名称。

不过，为了减少版本混淆，仍然建议修改版本发布者补充说明，例如：

- AgibotDisplay 修改版
- AgibotDisplay Extended
- AgibotDisplay Community
- AgibotDisplay by XXX
- 基于 AgibotDisplay 修改

这只是建议，不是强制要求。

---

## 关于品牌与非关联说明

请特别注意：

**使用 AgibotDisplay，不代表你获得了“智元创新”的品牌授权。**

你可以客观描述：

- 使用了 AgibotDisplay
- 基于 AgibotDisplay 修改
- 本项目采用 AgibotDisplay

但请不要误导性表述为：

- 智元创新官方授权字体
- 智元创新官方合作字体
- 智元创新联名字体
- 获得智元创新认证或背书

详见 [`TRADEMARK_POLICY.md`](./TRADEMARK_POLICY.md)。
